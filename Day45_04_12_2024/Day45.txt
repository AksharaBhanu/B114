String 
1. group of characters
2. specified within double quotes
3. it is a non-private data type
4. it is Class
5. it is present in java.lang package (we need not import it)
6. it is final class (we cant create sub class)
7. it implements Comparable interface -sorting
8. toString method is overridden--> it returns value instead of address
9. equals method is overridden
10. 2 ways to create String object
	String s1=new String("Bhanu");
	String s2="Bhanu";
11. to compare 2 Strings do not use == , use equals or equalsIgnoreCase
